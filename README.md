<h1>how to run</h1>
1. open <b>'babylon_bundling'</b> directory in terminal.<br>
2. type <b>'npm install'</b><br>
3. type <b>'npm run build'</b><br>
4. open <b>wpfbabylon.sln</b><br>
5. build solution.<br>
6. run
